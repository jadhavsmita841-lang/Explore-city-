# 🌍 Explore City

Explore City is a web application that helps users discover cities, tourist attractions, restaurants, hotels, and popular places. It provides an easy-to-use interface for exploring destinations .

## ✨ Features

- 🏙️ Browse popular citie
- 📍 View tourist attractions
- 🍽️ Find restaurants and cafes
- 🏨 Explore hotels
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- PHP

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/jadhavsmita841-lang/Explore-city-.git
   ```

2. Open the project folder.

3. Run `index.html` in your browser.
